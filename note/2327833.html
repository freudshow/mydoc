Debian安装compiz实现3D桌面效果

装好显卡驱动后就寻思着要实现3D桌面,而且可以使用dock.

1>测试机器有没有开启3D加速:

#aptitude install mesa-utils

#glxinfo|grep 'direct rendering' 如果显示direct rendering: Yes 则说明3D加速已经开启,否则可以能显卡驱动安装还存在问题吧(注:我在执行这条命令的时候,发现只有Root Terminal才有用,普通用户Terminal即使是通过su命令也无法显示出direct rendering: Yes,不知道是怎么一回事)

2>#aptitude install compiz

　　#aptitude install compizconfig-settings-manager

　　#aptitude install fusion-icon

　　#aptitude install compiz-fusion-plugins-main compiz-fusion-plugins-extra 注意如果不装这两个插件的话,到时可以设置的特将会少好多喔.

3>设置fusion-icon开机启动: System->Preference->Startup Applications->Add在弹出的对话框中写入如下内容

Name: 3D

Command: fusion-icon

Comment: 3D

4>注销后重新登陆就可以了

注: 可能会出现没有边框导致对话框不能移动的情况,这时候,右击那个蓝色的fusion-icon图标,把Compiz Options下面的那两个勾全部勾上就可以了.

5>至于dock网上有很多说的,我个人比较喜欢的还是Docky,感觉相当的简洁,而且该有的功能也都有了.具体安装可以通过软件中心也可以#aptitude install docky命令来实现.

6>装好dock之后可以把下面的panel先移除,把位子让给dock,整体感觉很不错的喔~
